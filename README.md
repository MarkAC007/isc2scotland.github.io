Official website for the ISC2 Scotland Chapter, built with Jekyll and deployed via GitHub Pages.

**Live Site:** [isc2scotland.github.io](https://isc2scotland.github.io)

## Local Development

```bash
# Install dependencies
bundle install

# Run local server
bundle exec jekyll serve

# Site available at http://localhost:4000
```

## Deployment

The site automatically deploys to GitHub Pages when changes are pushed to the `main` branch.
